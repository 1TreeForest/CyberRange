产品特性

视频上传

云点播提供多种上传 SDK，支持本地上传，URL 拉取和 API 上传等多种上传方式，在弱网的情况下仍旧可以有效保证上传速度和上传成功率，同时支持断点续传、大文件上传、以及文件多重备份功能。

全球存储

云点播支持全球 50+ 存储地域，提供冷/热备存储，支持源站迁移、媒资检索、媒资分类等能力，采用多种上传方式（就近上传、指定地域上传）保证用户更快，更流畅，更低延迟访问点播视频。

媒资管理

云点播的海量媒资信息支持批量管理和编辑操作，媒资信息多样包含字幕、雪碧图、截图、弹幕承载用户各种业务场景需求，同时云点播的媒资支持过期时间、禁播、分类便于用户更好的管理自己的媒资文件。

视频处理

针对海量多媒体数据，提供的云端转码和音视频处理服务。云点播支持 H.264/265/AV1 多种编码方式和极速高清、自适应码流、截图、内容识别、内容分析、内容审核等视频处理操作，云点播基于腾讯云音视频技术中台，提供给客户质量更高的视频处理服务。

视频加密

基于腾讯云终端和音视频处理技术，支持基于 HLS 的标准加密、私有协议加密和本地加密三种加密方案，有效解决视频盗播带来的经济损失。

全球加速

全国布局 CDN 加速节点超过 2000+ 个，覆盖电信、联通、移动以及长宽、广电等十几家中小型运营商；境外布局 200+个 CDN 加速节点，覆盖北美、欧洲、东南亚等 50+ 多个国家和地区，支持 CDN 加速业务无缝出海。

多终端播放器 SDK

云点播超级播放器支持多终端服务，支持多端数据上报和统计。云点播同时开源超级播放器 Adapter ，帮助用户无需耦合点播超级播放器就可以轻易集成云 PAAS 能力。

数据质量平台

基于云点播的云端一体体系，云点播提供基于超级播放器和云 PAAS 的数据质量平台，同时云点播集成任务管理、云监控等能力，帮助用户了解业务的上传、存储、转码、分发等能力的业务规模和突发情况。

直点一体化

基于直播类实时视频，云点播提供完整的一站式直点解决方案，包含直播录制、即时剪辑、伪直播、直播时移和直播降冷等能力，帮助用户更快捷的进行二次分发，引流提供商机。

产品架构

腾讯云点播（Video on Demand，VOD）为用户提供云端一体化音视频点播服务平台。用户可以通过本地上传/URL 拉取/API 上传/Web SDK/短视频 SDK/直播录制等多种上传方式将视频上传至云端，并对云端视频进行普通转码、极速高清、多码率转码、截图、视频加密、添加水印、识别等处理，处理完成的视频文件可通过腾讯云遍布全球的 CDN 节点完成加速分发。此外，结合云点播的超级播放器的多终端能力，提供完整可靠的数据质量和监控平台，同时云点播用户还可使用腾讯内容公众平台，小程序进行发布和推广，多分发路径使信息更易触达。
