怀旧经典 | 这些香港老电影值得你n刷（上）

迅雷电影
全网影评大咖都在这

上周末看到好朋友的朋友圈惊讶发现两人都不约而同地重温了一遍发哥、红姑钟楚红和哥哥张国荣的电影《纵横四海》。如果你还没看过，推荐一定要看。


当年的他们是那么年轻，在屏幕上散发着青春的活力。他们无疑是香港影史上最赞的三人行。可惜如今红姑不再演戏，哥哥已经走了多年。永远也不能再见他们同框。唯有想念的时候多刷几遍剧。

其实，香港电影鼎盛时期的许多电影至今仍值得n刷。

接下来两周影视汇，影小咖都会给大家推荐一些香港老电影，以下排名不分先后。


Ξ 1 Ξ

《英雄本色》

上映时间：2017-11-17(中国大陆) / 1986-08-02(香港)

导演： 吴宇森

主演： 周润发 / 狄龙 / 张国荣 / 朱宝意 / 李子雄

在这部经典电影里，发哥贡献了好几个经典镜头，“纸钞点烟”还在前阵子的《无双》里复现，勾起了无数人的回忆。

那个时候，张国荣在片子里的还是略显稚嫩。

影片里那些兄弟情义，江湖义气，不知道那个时候的你们能理解多少。心疼小马哥。


Ξ 2 Ξ

《暗花》

上映时间：1998-01-01

导演：游达志

主演：刘青云 / 梁朝伟 / 邵美琪 / 王天林 / 卢海鹏

《暗花》是银河映像于1998年出品的警匪动作片，主要讲述了腐败警察阿琛和杀手耀东之间的博弈，却不幸都沦为幕后老板洪先生的棋子的故事。

何为暗花？是指在黑市交易中，不露面的买主抛出的悬赏金。

市民刘先生和梁朝伟两大影帝同场飙戏，绝对过瘾。以下这个镜头，让小咖想起了之前的《摩天营救》，也有这么一个情节设置。


Ξ 3 Ξ

《枪火》

上映时间：1999-11-19(香港)

导演：杜琪峰

主演：吴镇宇 / 任达华 / 黄秋生 / 吕颂贤 / 张耀扬

一部黑帮片，被称为银河映像最经典的作品。节奏干净利落，情节耐人寻味，看完不妨到迅雷电影搜索枪火看看别人怎么分析。


Ξ 4 Ξ

《一个字头的诞生》

上映时间：1997-03-14(香港)

导演：韦家辉

主演：刘青云 / 吴镇宇 / 李若彤 / 徐锦江 / 张达明

阿狗（刘青云饰）是三十多岁的黑社会马仔，他面临着一个艰难的抉择，要么继续浑浑噩噩的在黑道上混下去，要么轰轰烈烈的大干一场。影片平行展开两个不同的命运，主人公的一念之间人生的走向就完全不同了。


Ξ 5 Ξ

《香港制造》

上映时间：1997-10-09

导演：陈果

主演：李灿森 / 严栩慈 / 李栋全 / 谭嘉荃 / 林洁芳

港式青春片，有着不一样的疼痛。

边缘少年中秋，是一个家境惨淡的小混混，但为人却很豪爽讲义气，将总被人欺负的弱智男孩阿龙（李栋泉）当作自己的亲人“罩”着。他爱上了身患绝症的少女阿屏，命运从此变得不一样。所有的一切，都逼得他越来越绝望，他该何去何从？


Ξ 6 Ξ

《倩女幽魂》

上映时间：2011-04-30(中国大陆) / 1987-07-18(香港)

导演：程小东

主演：张国荣 / 王祖贤 / 午马 / 刘兆铭 / 林威

这部估计不用我多介绍了，还有谁没看过的，举下手！

女神王祖贤，真的太美了。


顾盼生辉


Ξ 7 Ξ

《秋天的童话》

上映时间：1987-07-16

导演：张婉婷

主演：周润发 / 钟楚红 / 陈百强 / 黄淑仪

发哥和红姑合作的一部爱情片，讲的是一个美女留学生和一个唐人街打工仔的爱情故事。

《秋天的童话》在前些年香港电影十佳电影里排名第二，排名第一的是《英雄本色》。

看到有人说

“反正我第一次看完秋天的童话后就再也不看少男少女的爱情电影了，因为电影中发仔这个三十岁男人的初恋宛如一瓶醇香的红酒永远让我沉醉当中”


Ξ 8 Ξ

《金枝玉叶》

上映时间：1994-07-23(香港)

导演：陈可辛

主演：袁咏仪 / 张国荣 / 刘嘉玲 / 陈小春 / 曾志伟

这是一部爱情喜剧。张国荣饰演的音乐家顾家明，刘嘉玲饰演的玫瑰，同时爱上了女扮男装的林子颖（袁咏仪饰）。性别的错乱，导致笑料不断。

有人说，三人最好的相处方式，不应该是刘嘉玲和袁咏仪在一起，然后哥哥在旁边英俊地弹钢琴吗？

也真是皮了。


Ξ 9 Ξ

《新龙门客栈》

上映时间：2012-02-24(中国大陆) / 1992-08-27(香港)

导演：李惠民

主演：张曼玉 / 林青霞 / 梁家辉 / 甄子丹 / 熊欣欣

宦官专权的明朝中叶，东厂曹少钦（甄子丹）假造圣旨杀害了忠良杨宇轩，并想斩草除根杀死其后代。侠女邱莫言（林青霞）、江湖义士贺虎等人冒死救出忠良后代后，逃至边关沙漠里的龙门客栈，与杨宇轩部下周淮安（梁家辉）会合。

龙门客栈实则黑店，老板娘金镶玉（张曼玉）利用乱世，以美貌和风骚 作诱饵杀人越货聚敛钱财，但她却为英俊潇洒的周淮安动了真心。

电影在遵循传统武侠形制的基础上有所突破，开一派新武侠的景致，被誉为香港新派武侠的起点，堪称武侠电影的一座丰碑。


Ξ 10 Ξ

《东邪西毒》

上映时间：1994-09-17(中国大陆)

导演：王家卫

主演：张国荣 / 林青霞 / 梁朝伟 / 张学友 / 张曼玉

《东邪西毒》里，小咖最喜欢的一句台词是

“很多人想知道沙漠的那边是什么，其实沙漠的那边是另外一个沙漠。”

丧到极致，但是却又充满了哲理。


最后，影小咖要特别强调一下

强烈建议尽量看原汁原味的粤语版，整个feel都不一样的。
