2018大中华区人力资源服务机构品牌

英格玛集团成立于2002年，是中国人力资源外包服务机构；集团致力于成为中国人力资本经营服务专家，让人力资本实现最大价值，从而推动社会发展，并让我们的世界变得更加美好。

集团主营业务包括：制造业外包、服务业外包、互联网外包、管理咨询与培训、招聘服务、英格玛金融、国企央企服务、汉普法务外包、EngmaIntec英特科等多个专业服务领域。集团先后扎根于苏州、上海，形成双总部格局，旗下现拥有140多家分支机构（包含苏州总部 / 上海总部 / 北京区域总部 / 深圳区域总部 / 杭州区域总部 /广州 / 重庆 / 成都 / 武汉 / 合肥 / 西安 / 沈阳 / 无锡 / 常州 / 烟台 /天津 / 南京 / 长沙 / 济南 / 郑州 / 大连 / 哈尔滨 / 贵阳等城市）；集团现有管理团队1000多人，外派雇员100000多人，年培训及供应人才能力超过20万人。
