BT 种子资源网站分享

2017-10-20 13759点热度 16人点赞 0条评论

提到种子，相信大家肯定都不陌生，特别是喜欢看电影和玩游戏的小伙伴，会经常使用种子来下载资源。

由于 BitTorrent 协议的特殊性，经常被用来下载一些非法版权的资源，很多地方都在封锁 BT 资源网站和服务器。

不过完全封掉是不可能的，很多国家并没有法律规定 BT 是违法的，而且 BT 可以显著减轻源下载服务器负担，很多 Linux 发行版就使用 BT 提供安装镜像。

说了这么多，那么有哪些好用，资源丰富的 BT 网站呢？

这篇文章就给小伙伴们分享一些 BT 资源网站

Bittorrent 下载工具

工欲善其事，必先利其器。

所以呢，一款优秀的 Bittorrent 下载工具是必不可少的，下面就给大家介绍几款下载工具。

迅雷

迅雷，大家一定不陌生，应该是国内使用人数最多的下载工具。

虽然常被大家诟病下载限速，不过它确实是个不错的下载工具，下载某些种子资源的时候，速度不输给任何一个下载工具

这里给大家分享迅雷极速版的绿色优化版，比起迅雷9的内置浏览器，极速版可谓是相当的好用。

链接: https://share.weiyun.com/5Or21P0 密码: mivmcn

BitComet

BitComet 又名 比特彗星，是一款免费的 HTTP/FTP/BT 下载软件。

它的特性包括同时下载，下载队列，从多文件种子中选择下载单个文件，快速恢复下载，磁盘缓存，速度限制，端口映射，代理服务器和IP地址过滤，种子市场等。最新的版本中加入了公用DHT网络的支持和反吸血（针对迅雷）设定。

推荐小白用户使用这款软件，操作简单。

官方网站：www.bitcomet.com

qBittorrent

qBittorrent 是一个跨平台的自由 BitTorrent 客户端，其图形用户界面是由 Qt 所写成的。使用 libtorrent 作为后端。是由 Christophe Dumez 在贝尔福 - 蒙贝利亚尔理工大学时开始开发。从2006年3月开始， qBittorrent 进行积极的开发。

qBittorrent 试着提供比其他常见的相同类型的程序更轻巧，但更强大、更直觉、以及更有吸引力的外观。qBittorrent 提供一个类似 μTorrent 但开放源代码且跨平台的选择，并加入了一个类似流的功能，让用户可以一边下载，一边观看该视频。qBittorrent 拥有的功能大略可与 Vuze 等其他较流行的 BitTorrent 客户端相比，但却不需要 Java 虚拟机。qBittorrent 在搜索种子文件时只需要用到 Python，而不像其他客户端（如 Deluge 或 BitTornado）需要用到 BitTorrent 协议。

如果不喜欢 µTorrent 的广告，可以在 Windows 尝试这款软件，小山个人觉得十分好用。

官方网站：www.qbittorrent.org

µTorrent

µTorrent（或称为 uTorrent 或 microTorrent；缩写 µT 或 uT）是一个用 C++ 语言写成，于 Microsoft Windows 以及 Mac OS X 操作系统下运行的 BitTorrent 客户端免费软件，并且经过许多语言的本地化。此软件设计为在运行时使用较少的系统资源（与 BitComet 和 Vuze 比较）。µTorrent 因为其特别的设置、表现、稳定性和对老旧硬件以及旧版 Windows 的支持受到许多一致的好评。µTorrent 从2005年首次发表至今持续开发中。

在安装的时候需要注意，安装包会询问你是否安装 XXX 第三方软件，注意不要安装。

* 这款软件内包含广告

官方网站：www.utorrent.com

Transmission

Transmission 是一款开源的 Bittorrent 下载工具，特点是一个跨平台的后端和其上的简洁的用户界面。

Transmission 被众多 Linux 发行版，包括 Ubuntu、Mandriva、Linux Mint、Fedora、Puppy Linux、openSUSE 选作默认 BT 下载工具；Imageshack 的服务使用其技术。

我并不推荐在 Windows 平台上使用这款工具，因为它相比于其他工具没什么优势，如果你有 OpenWrt 路由器或者是树莓派之类的，可以在其上面安装 Transmission。

官方网站：transmissionbt.com

注：以上部分软件简介摘自维基百科

全能

以下网站适合搜索任何类型的种子

The Pirate Bay：又称：海盗湾，是一个专门存储、分类及搜索Bittorrent种子文件的网站，并且号称是“世界最大的 BitTorrent tracker（BT种子服务器）”，提供的BT种子除了有自由版权的收集外，也有不少被著作人声称拥有版权的音频、视频、应用软件与电子游戏等，为网络分享与下载的重要网站之一。相信不少小伙伴都不陌生，号称全球最大的 BT 网站，其资源是相当的丰富，也被很多国家禁止访问。

RARBG：成立于 2008 年保加利亚的一家 BT 资源网站，主要以电影和电视节目为主，而且资源多数为 1080、4K、3D、蓝光级别，由此也遭受了多个国家的封锁。喜欢看电影的小伙伴应该对这个网站不陌生，有很多电影的蓝光资源，当然，其他资源也可以在这里找到。

Torrentz2：Torrentz 的克隆，它是一个快速强大的种子搜索引擎，集合了数十种种子资源网站的搜索结果，截至目前，它已经从 82 个网站索引了 2.5 亿个种子。这个网站支持中文搜索，可以搜索一些国内资源。

Rutracker：一家位于俄罗斯的 BT 资源网站，拥有活跃的文件共享社区，RuTracker.org 已成为俄罗斯最受欢迎的网站之一。这个网站的很多文件需要注册登陆才可以下载。

ETTV：也是一家很不错的 BT 资源网站，跟 RARBG 配合起来，基本上可以该有的资源都能有。

电影

以下网站适合搜索电影类型的种子

RARBG：上面已经介绍过了，绝对是下载电影首选。

ETTV：不多说了。

YTS：以前的 YIFY 网站的克隆，该网站最大的特点是你能以最小的文件体积下载到 720P、1080P 和 3D 电影，正常的 3D 电影文件起码 20G+，这里，只需要 2G+，当然，影片质量肯定会受损，不过对于一些网速不好的小伙伴，又多了一种选择。

Cine Calidad：一家西班牙电影网站，该网站最大的优点在于，电影可以在线观看或者通过 MEGA 和 MediaFire 网盘下载，当然，同样支持 BT。不过由于服务器位于国外，如果想流畅的在线观看，你可能需要一个高速代理，或者。。。IPv6 网络。

乐赏：国内的一家资源分享论坛，由 HQC影视小组 和 GBT游戏小组 组建而成，以分享电影、影视剧、游戏为主，很多国内的电影和电视剧都可以在这里找到。

中国蓝光网：一家国内网站，资源整合能力不错，如果你看不懂英语，或者没有代理工具，这个网站或许适合你。

动漫

以下网站适合搜索动漫类型的种子（由于小山不是二次元，所以。。。我并不知道他们有什么好的。）

Nyaa：一家侧重于东亚（中国、日本及韩国）多媒体资源的 BT 资源网站，它曾是世界上最大的动漫专用种子索引站。真正的 Nyaa 已关闭，此为镜像站。

Sukebei Nyaa：Nyaa 成人片段的镜像站，你们懂的。

HorribleSubs：一个迷你小组，拥有大量的动漫系列，可以下载 480P、720P、1080P 视频格式。他们自己翻译、编码和编辑动漫节目。该网站提供多种下载方式：种子、网盘、XDCC（一种通过 IRC 下载文件的方式）。当 Nyaa 关闭时，这个网站成为许多 Nyaa 用户的替代品。

游戏

以下网站适合搜索游戏类型的种子

IGGGAMES：是国外的一家游戏下载网站，游戏资源多。提供 BT 和 网盘下载，而且都是很优秀的国外网盘。

Skidrow & Reloaded：是一家专注于游戏下载的网站，除了 BT 下载外，它还提供了多种网盘链接下载，同时还有每个游戏发行版的下载量。这个网站很不错，热门大作基本都可以找到，而且还有很多游戏的 MOD 可以下载。

GamesTorrents：是一家专门提供游戏下载的西班牙种子资源网站，它不仅提供 PC 游戏，还提供多种语言的 PSP、PS3、PS4、NDS、Wii 和 Xbox 360 平台的游戏可供下载。

乐赏：上面已经介绍过了，这里的游戏资源也很丰富。
