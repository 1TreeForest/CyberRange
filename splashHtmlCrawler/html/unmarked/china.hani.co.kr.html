韩国喜剧电影《极限职业》搞笑至极

登录 : 2019-02-08 10:20
  • 페이스북
  • 트위터
  • print 프린트
  • Smaller font size 글씨크기 크게
  • Larger font size 글씨크기 작게

继《与神同行》后，史上第23次吸引千万观众

《7号房的礼物》之后，时隔6年出现喜剧电影突破千万
并非人气明星的5名主演的配合尤为出彩
融合了组织生活的疲惫与个体户的悲哀

图为电影《极限职业》中的一个场面。 (图片来源：CJENM)

“到目前为止还没有这样的电影。对，就是千万票房电影《极限职业》。”

喜剧电影《极限职业》上映半个月后，2月6日观影人数突破了千万。这是继去年8月突破千万票房的《与神同行-因与缘》之后史上第23部千万票房电影。同时也是继《七号房的礼物》之后，时隔6年突破千万票房的喜剧电影。《极限职业》突破了从去年下半年开始持续的韩国电影低迷状态，成为今年首部千万票房电影，被评价为为韩国电影市场开了绿灯。

6日，该电影的投资发行公司CJENM正式发布报道资料称，“电影《极限职业》上映15天后的6日下午12点25分，观众人数突破千万”，“这是继历代千万电影中《鸣梁》（12日）、《与神同行-因与缘》（14日）之后，创下的第三快纪录”。

图为电影《极限职业》中的一个场面。 (图片来源：CJENM)

《极限职业》的旋风从初期开始就展现出了与众不同的气势。相比于上映第10天即1日上映的《国际市场》、《辩护人》、《光海，成为王的男人》等其他千万票房电影，该片以极快的速度突破500万观众。从春节假期开始的2日开始，平均每天吸引100万名以上的观众。之后于2日、3日、4日、5日分别突破600万、700万、800万、900万。

有分析认为，《极限职业》的票房很大程度上得益于喜剧题材的特性。该片讲述了面临解散危机的毒品班刑警5人帮为扫荡毒品组织而伪装创业的炸鸡店后一跃成为美食店并名声大噪的故事。CJENM相关人士表示，“该片的优点是从头到尾一直带给观众笑容。感觉非常开心、非常轻松，非常符合观众想要开始新的一年的要求”。实际上在春节期间，喜剧电影也占优势。《7号房的礼物》（2013年）、《奇怪的她》（2014年）、《朝鲜名侦探：奴隶的女儿》（2015年）、《共助》（2017年）等横扫春节长假。

去年下半年开始上映的重磅电影给疲惫的观众带来了“黏糊糊的乐趣”，也渗透着男女老少无论是谁都能产生共鸣的情绪，这也是电影的一大优势。评论家郑志旭（音）表示，“《国家破产之日》、《毒品王》、《Swing Kids》等去年年末在大银幕上映的电影都是讲述沉重的时代氛围的电影，而这部作品既简单又轻快，融入了小市民的朴素情感”，“在寸铁杀人的台词中融入了警察的悲欢、后辈们被组织生活压榨的辛酸以及无法预知明天的个体户的悲哀”。

图为电影《极限职业》中的一个场面。 (图片来源：CJENM)

最让人惊讶的是，《极限职业》的千万票房并不是一个明星演员的票房号召力，而是由5位主演均衡的活跃和配合拼凑出来的壮举。郑评论家表示：“除了柳承龙以外，其他演员都不是A级明星。这证明了扎实的剧本和演员们的默契配合能够拍出千万票房的电影”。

虽然是一年中韩国电影界正式竞争的春节季，但没有强有力的竞争作品这一点也对《极限职业》的票房产生了积极的影响。某投资发行公司相关人士表示，“《极限职业》从剧本阶段开始就在忠武路以超搞笑的电影闻名，因此被其他制作公司或投资发行公司选为了制定对阵表时最应该先避开的电影”。实际上，与《极限职业》对抗的电影只有《逃组》，但它只吸引了130万名观众。被称为詹姆斯•卡梅隆导演力作的好莱坞大片《阿丽塔：战斗天使》避开了《极限职业》，于本月5日上映。

关键要看《极限职业》的气势能延续多久。据悉，该片的损益平衡点为230万人，这已经是制作费的近5倍。CJENM相关人士表示，“《极限职业》至今在观影人数上仍处于上升趋势，周末过后将突破1200万。期待最终人数能达到1400万左右”。

刘善姬 记者
