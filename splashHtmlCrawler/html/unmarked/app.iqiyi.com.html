爱奇艺影音用户帮助

如果这些信息不能帮助您解决问题， 请给我们 反馈

1.如何解决windows XP 系统下出现的画面倒立现象？

请在设置中勾选“翻转播放”选项，如下图所示（注：若您不是XP系统也出现画面倒置请通过反馈与我们联系）

2.播放器窗口显示黑屏或空白怎么办？
解决办法：
1、检查网络连接状况，确认网络连接正常。
2、建议您下载或更新Windows Media Player至最新版本。 立刻下载安装最新版本 Windows Media Player
3.版权受限是什么？为什么我会看不了？
  • 奇艺所有的视频都为正版内容，由于版权的原因，有些视频只能在特定的区域能观看。如果您的所在地IP不在某视频的版权授权范围内，就会导致您无法观看该视频。
4.一直显示“视频加载中，请稍候“怎么办？
解决办法：
1、持续10S之内的加载是很正常的，请稍候。
2、如果视频加载时间过长，您可以先检查自己的网络情况：能否登陆其他网站，带宽因素（带宽越小，加载时间越长）。
3、如果网络没有问题，您可尝试在载入页面右侧“清晰度”菜单中，选择“高清”或“流畅”，应用设置后爱奇艺影音会为您重新播放。
4、也可先选择收看其他精彩视频，察看是否节目源问题。
5.画面卡，提示“连接不到服务器”或处于“正在连接到服务器”的提示状态不动？
解决办法：

1、连接不上服务器可以尝试向后拖动进度条。
2、检查网络连接状况，确认网络连接和网速是否正常。
如果您正在使用Windows XP SP2操作系统，需使用过XP SP2突破连接限制补丁。该补丁会在每次升级windows以后失效， 需重新运行（点击下载XP SP2突破连接限制补丁）。
如果您正在使用卡巴斯基防火墙等安全软件，请在播放视频时关闭“网络实时监控”功能。
如果您是内网用户，请使用端口映射或打开UPNP支持。
3、当网速慢或者网络连接不稳定的时候会出现画面卡的情况，用户可以升级带宽，或者避开高峰期观看视频。
如果您正在使用迅雷下载等P2P下载软件，或同时打开其他视频页面，建议关闭类此程序。
4、用户也可以在播放器“清晰度”中选择“流畅”，流畅的带宽要求要比高清低。
高清需1M，超清需2M以上带宽支持才可流畅播放。
5、您的IP地址，或许正处于我们服务器布署的一个死角。请将您的IP地址提供给客服人员，我们会努力让每一个人都能享受到流畅的视频服务。 提交入口>>

6.视频有声音但画面停止，或图象正常但无声音？
解决办法：

1、您可以检查是否已关闭硬件加速功能，音频播放设备可用、已开启并调试到合适音量。
2、节目源有可能存在问题，请等待技术人员恢复。
3、有些影片受片源影响可能出现音量较小的情况。

7.超清是什么意思？我为什么看不了超清？

“超清”是奇艺提供的更清晰的观看体验，观看“超清”需要网速超过2M以上。奇艺会有选择的提供超清码流，保证超清码流的最佳观看体验。

8.怎样才能在爱奇艺影音中快速从上次看到的时间点继续观看？

爱奇艺影音页面上方的“播放记录”将记录您上次观看的视频节目的名称，以及您上次看到的时间点，点击“继续观看”就能直接从观看您上次停止的地方继续观看，免除您忘记集数、频繁拖动进度条寻找上次看到哪儿的困扰。

9.如何将播放器固定在页面最前端显示？

爱奇艺影音提供了方便的播放器置顶功能，请点击播放界面上方导航条右侧的置顶图标

点击后即可将播放器固定在页面最前端显示。

10.如何在Windows XP SP2操作系统下使用爱奇艺影音？

如果您正在使用Windows XP SP2操作系统，需使用XP SP2突破连接限制补丁。该补丁会在每次升级windows以后失效，需重新运行。 点击下载 XP/SP2突破连接限制补丁.

11.如何解决win7系统下播放画面的锯齿现象？

如果您使用的是win7系统且播放画面会有锯齿现象，请对桌面组合进行重新设置。 右击我的电脑——属性——高级系统设置——高级——性能选项——启动桌面组合（默认是勾选的，请设置不勾选）

12.怎样选择合适的视频渲染器？
如果您观看视频时，出现模糊、马赛克，放大时失真等现象，建议您在播放设置中更改视频渲染器，以获得最佳观看体验。
13.如何选择离线加载多集视频(批量下载)？

鼠标在想要离线下载的视频上悬停，点击【离线观看】，多集的视频就会显示选择的界面，勾选想要下载的集数，然后点击【立即加载】即可.
