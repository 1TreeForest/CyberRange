安装要求

底线 PC 硬件要求

  • 奔腾双核 E214 1.6GHz、奔腾双核移动式 T2060 或奔腾赛扬 E1200 1.6GHz
  • 2GB RAM 1.2 GB 可用硬盘空间
  • USB 端口，1.1 或更高，用于连接 USB 扫描器
  • 显示器底线分辨率 = 1024 * 768 像素

注意：要使用 123Scan，必须配备 Windows 计算机和鼠标。123Scan 不支持使用触摸屏。

安装

  • 可以从“开始”屏幕启动 123Scan
