用PSP看电影
以下内容适用自制系统 3.71 M33及更高系统
∴ 前言
返回

　　作为一部拥有4.3英寸16：9的宽大屏幕数码娱乐掌机，很多人用PSP来作为一个掌上的视频播放设备。而PSP的大屏幕与高发色数的屏幕也让它在同价位的视频播放器面前卓尔不群。

本节内容
·PSP支持的视频格式(初级) ·如何分辨视频格式(初级) ·MP4与MP4-AVC(初级)
·MP4的播放方法(初级) ·PMP的播放方法(中级) ·自己动手转换视频(高级)
∴ PSP支持的视频格式
返回

　　目前能够在PSP上播放的视频格式如下。

　　MP4　扩展名：MP4——在PSP发售之初即支持的一种视频文件格式，采用MPEG-4编码格式，其分辨率一般为320×240。在PSP上可以拉伸到全屏播放。

　　MP4-AVC 扩展名：MP4——PSP于2.0版之后新支持的视频格式，采用H.264编码格式，分辨率同原始MP4文件相同，在PSP上可以拉伸到全屏播放。视频的流畅以及清晰程度上优于普通MP4格式。由于PSP具备H.264硬件解码,播放AVC编码的视频更为省电。

　　AVI-MJPGE 扩展名：AVI——PSP固件高于2.80版之后新增支持视频格式，要求视频的算法为MJPGE。该格式视频文件的体积比较大，非主流视频格式。

　　MP4-AVC （480×272）扩展名：MP4 （主流）——PSP固件3.03OE自制系统之后所支持的视频格式，压缩算法与MP4AVC相同，分辨率提高到与PSP屏幕分辨率相同的高度。可实现全屏播放效果。PSP具备H.264硬件解码,播放AVC编码的视频更为省电。

　　MP4-AVC （720×480）扩展名：MP4——PSP于3.30系统之后新增支持，基本上与之前的MP4AVC相同，但是将分辨率扩展到了720×480。由于视频本身分辨率高于PSP屏幕分辨率，所以本格式视频的清晰程度同UMD Video，缺点是体积增加了不少。PSP具备H.264硬件解码,播放AVC编码的视频更为省电。

　　PMP 扩展名：PMP——通过安装第三方软件所支持的视频格式，视频编码方式采用Divx、Xvid，分辨率通常为480×272。PSP需要通过PMP MOD或PPA等专用播放软件来观看。

　　PMP-AVC 扩展名：PMP （主流）——通过安装第三方软件所支持的视频格式，视频编码采用x264。具有低码率高质量的优点，PSP具备硬件解码H.264,播放PMP-AVC更为省电。

　　在PSP发售之初即支持的一种视频文件格式，采用MPEG-4编码格式，其分辨率一般为320×240。在PSP上可以拉伸到全屏播放。

　　不过，从清晰度和文件大小考虑，目前流行的实际上只有两种：MP4-AVC(480×272)以及PMP-AVC。其中MP4-AVC能够直接使用 PSP自带的播放功能播放，而PMP-AVC则需要安装自制软件才能播放。

∴ 哪儿获得电影
返回

　　获得电影的方法有以下几种，但由于版权原因我们强烈建议玩家通过正规渠道购买正版电影。

　　1、通过PSP主题网站下载PMP或者MP4格式的电影。目前，比较有影响力的PSP主题网站均有提供PSP专用格式电影下载，我们推荐使用电玩巴士BT频道提供的PSP电影。

　　2、通过电影下载站下载电影，并自行转换为PSP专用的MP4或者PMP格式。这里我们推荐使用迅雷下载 http://www.xunlei.com/。

　　3、购买PSP电影光盘，在国内各大中小城市内均有电脑软硬件交易市场，在这些地方购买PSP电影光盘即可，一般一张DVD盘包含8-10部电影，因电影容量不同而数量有所不同。

∴ 如何分辨格式
返回

　　从专门的网站上将PSP专用电影下载过来之后，首先分辨该视频是MP4（包括MP4 -AVC）还是PMP（包括PMP-AVC），一般来说可以从文件的扩展名中看出来。如果你的电脑上看不出文件的扩展名，可以按“显示扩展名” 的方法进行设置。当然，大多数情况下，你下载电影的时候便已经从网站获知了视频的具体格式。

　　MP4格式视频的扩展名为.MP4、PMP格式视频的扩展名为.PMP

　　PSP针对不同的视频格式拥有不同的播放方法。由于AVI-MJPGE格式的视频文件体积巨大而效果却不佳，所以目前几乎没有网站会提供此种格式的视频下载。另外由于H.264编码相对于MPG－4的巨大优势，一般来说，如果你下载的电影是近期发布的话（2007年内及之后），基本上可以将它们看作是 AVC格式的电影（MP4 AVC或PMP AVC）。

∴ MP4与MP4-AVC
返回

　　由于MP4与MP4-AVC的播放方法并不相同，所以我们仍需对它们进行区分。二者的后缀名都是MP4，一般下载的时候都会说明，是MP4还是AVC，有的名字会已经改好，M4Vxxxxx就是MP4，MAQxxxxx就是MP4- AVC了。

　　但如果遇到没有特别说明的MP4文件的话，我们也可以在PC上面区分MP4和MP4-AVC。具体方法如下：用暴风影音等播放器播放你要鉴别的视频文件，然后在播放器选择“文件” -> “属性” -> “详细资料”，如果是AVC，MPEG4 Video后面会显示H264（也有可能显示AVC，取决于转换方法的不同）。

∴ MP4播放方法
返回

　　1、首先将MP4文件拷贝到记忆棒内。由于官方的要求，这两种视频格式对文件名以及文件放置的位置都有一定的要求。正确的放置和命名方法如下。

　　2、提示：文件名和视频文件一样，后缀为.THM的文件为预览文件，实际是一张160x120的JPG格式图片，非必须的文件，和视频文件放在一起就可以预览了。

　　一些工具会在影片转换时同步生成预览图片，如果你有特殊需要的话可以找一张影片的相关图片，随便用工具调整为160x120大小并保存为JPG格式图片，再在电脑上改后缀名为THM，文件名和影片名一样，按上述方法放置就可以了。

　　将文件放置好以后，回到PSP主界面选择VIDEO -> Memory Stick，就可以播放了。播放中的操作很简单，并且按△后有对影片播放详细操作。

　　补充：在最新的系统中已经支持了存放于VIDEO文件夹（记忆棒根目录）下的MP4文件的播放，根据官方的说明，存放于VIDEO文件夹下的任意格式的MP4都能够正常播放，不需要再费神设置文件名和文件夹。但是在我们电玩巴士编辑部的实际测试中发现VIDEO文件夹并不能解决所有问题。由于PSP早期对视频的限制，当时在PSP上使用的MP4很多都是经过特殊处理的，这些方法转换的MP4（含MP4 AVC）并不能在VIDEO文件夹下正常播放，必须放到MP_ROOT下面。又由于操作上的惯性，现在一些转换组转换MP4的时候依然采用的是早期的转换方法，而没有采用最新的规格，导致我们依然需要按照MP_ROOT的规则设置视频。

∴ PMP的播放方法
返回

　　1、由于PMP格式的特殊性，所以目前在电脑上没有比较方便的区分方法。不过无论是 PMP还是PMP-AVC，在使用最新的播放器PPA后，它们的播放方式已经没有任何区别了，从这点说，我们可以不考虑它们的区别。

　　要播放PMP，首先我们需要安装PMP专用播放软件PPA（PMPlayer Advance）。下载该软件，并将PPA3xx目录拷贝到记忆棒中的PSP/GAME目录下。

　　将PMP文件拷贝到记忆棒根目录的Video下，打开PSP，在游戏列表中找到PMPlayer Advance。

　　按○进入可看到主界面

　　PPA使用说明：

　　△ ——退出播放
　　○ ——调出播放信息OSD（播放信息已移到画面上方，不再影响字幕显示）
　　□ ——暂停/继续播放
　　左/右 方向键——快退/快进
　　上/下 方向键——音量增/减(会引起失真)
　　L/R ——亮度增/减
　　SELECT——音轨切换
　　START ——画面比例切换
　　×+L ——外挂字幕切换/关闭
　　×+R ——循环播放功能开启/关闭
　　×+左/右 方向键——高速快退/快进
　　×+上/下 方向键——画面缩放
　　对于喜欢全屏的朋友，也可以实现4：3和2.35：1的不变形全屏播放，再结合外挂字幕功能，基本完美。

PSP播放电影常见问题

问：PSP支持哪些格式的电影？
答：PSP官方系统和自制系统对于电影的格式支持程度并不一样，PSP官方系统仅支持MP4及AVI格式电影，而自制系统在支持官方系统所有格式之下还支持PMP格式电影，一般地版本越高支持电影的格式越丰富，所以强烈建议升级到最新的系统。

问：PSP播放电影耗电么？
答：PSP播放电影因为电影本身格式的不同耗电量也不同，一般地说，播放AVC格式的电影最为省电，充满电的PSP播放这种格式的电影大约能持续4-5个小时（由于客观因素影响较大，以实际情况为准）。

问：为什么把电影放到VIDEO里却不能看？
答：PSP电影不是放到VIDEO里，PSP用电影应该放到“MP_ROOT”这个文件夹下的“100MNV01”（MP4格式用）和“100ANV01”（AVC格式用）这两个子文件夹里，并且MP4影片名字要改成M4VXXXXX.MP4（XXXXX带表任意的5位数字）AVC影片要改成MAQXXXXX.MP4才能正常播放。

问：命名是MP4格式的电影为什么不能播放？
答：出现这个问题有以下两种可能：1、PSP不是所有的MP4格式影片都可以播放，事实MPEG4这个格式还要分成几种不同的格式，具体情况请看本页面里的“MP4与MP4-AVC”章节；2、你下载的可能是AVC格式的影片，因为AVC格式的影片的后缀名也为.MP4，这种格式需要官方系统2.0以上及自制系统才能正常播放，具体播放方法请查看本页面“MP4播放方法”章节。

问：我的小P如何才能播放RMVB格式的电影？
答：这个很抱歉，目前小P不支持播放RMVB格式的影片。

问：我的小P如何播放AVI格式影片？
答：目前最好的方法是用转换器转换成MP4格式来看，具体转换方法请查看本页面“自己动手转换PSP电影”章节。

问：什么是全屏AVC？怎么播放？
答：因为SONY对自己UMD电影进行了一项特殊的保护措施，导致所有PSP均不可以播放480*272分辨率的电影，而OE系统为我们解开了这个封印，3.02 OE-B以后的OE系统均可以播放480*272分辨率的AVC电影，这种分辨率的视频称之为全屏AVC。

问：PMP格式是怎么回事？小P如何才能播放？
答：PMP全名为“Portable Media Player“意思为便携式媒体播放器。PMP是目前PSP上公认的清晰度最强的电影格式，需要通过给安装自制软件才能播放，比较常用的软件是psp media player，安装好后把PMP电影放到VIDEO里（注意是PSP下的VIDEO子文件夹，不是根目录的MP-ROOT里）然后在GAME的最后一项里运行psp media player这个软件就可以了，清晰度非常理想。

问：PSP如何播放WMV格式的电影？
答：这个很抱歉，目前小P不支持播放WMV格式的影片，只能转换成MP4格式的电影后方可播放，具体转换方法请查看本页面“自己动手转换PSP电影”章节。
