ZY-Player – 可播放12家在线视频的跨平台开源视频播放器

Windows linux macOS 2020/05/30 青小蛙 23
(16票)
Loading...

ZY-Player 是一款开源的跨平台视频播放器，拥有 Windows、macOS、Linux 客户端，支持直接播放12家在线视频网站的视频内容。可扫码在微信播放。@Appinn

来自发现频道。

这是一款集合了 12 家在线视频网站内容的视频播放器，包括：

  • 最大资源网
  • OK资源网
  • 麻花资源
  • 哈酷资源网
  • 等等青小蛙不太听说过的视频网站

ZY-Player 简单粗暴，启动后就直接显示资源列表，可以在左上角切换视频源，点击视频名称就进入选集页面，可直接播放以及复制下载按钮。

注意 ZY-Player 不提供下载功能，仅提供下载链接。当然最方便的还是直接播放。

资源很丰富，但注意：有个别资源包括🔞内容，请家长朋友们注意了，未满18岁的同学也请（注意不是那种🔞内容），啊，你们谁看过了来解释下是什么内容啊，青小蛙表示说不清楚。

可在微信播放

ZY-Player 可以在播放的时候生成二维码，通过微信扫码之后即可直接在微信播放。

下载 ZY-Player

可以在 GitHub 下载 ZY-Player，一个国内搬运下载地址（Windows、macOS、Linux[snap 格式]）。需要事先手动安装 ffmpeg。
