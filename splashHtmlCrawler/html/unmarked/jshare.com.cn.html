  • 快捷键

    快捷键风格同 Sublime，同时支持 Emmet 特性，常用快捷键：

    全局快捷键
    • Ctrl + Enter：运行
    • Ctrl + ←/→：折叠左侧菜单
    • Ctrl + O：打开添加常用资源
    • Ctrl + Z：撤销所有编辑器
    • Ctrl + U：恢复所有编辑器
    • F11：最大化运行结果
    • ESC：关闭添加常用资源
    编辑器快捷键
    • Ctrl + Enter：运行
    • Ctrl + /：注释代码
    • Ctrl + H：格式化代码
    • Ctrl + Z：撤销
    • Ctrl + Y：恢复
    • Tab：快捷生成代码
    • Ctrl + L：选中当前行
    • Ctrl + Q：折叠当前代码
    • F11：最大化当前编辑器
