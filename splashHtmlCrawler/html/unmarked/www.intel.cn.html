如何在 Windows® 10 和 Windows 11* 中安装英特尔® 显卡驱动程序

Close Window

文档

安装与设置

000005629

2022 年 01 月 26 日

安装计算机制造商提供的最新显卡驱动程序，或使用以下方法之一从下载中心安装最新的显卡驱动程序。

更多信息请参阅为什么应该更新显卡驱动程序。

注 我们建议您先与计算机制造商确认，并使用他们提供的驱动程序软件，以避免潜在的安装不兼容问题。英特尔提供显卡驱动程序的通用版本，适用于常规用途。计算机制造商可能对图形驱动程序软件或软件包功能作了更改、加入了自定义配置，或作了其它更改。

方法 1：使用英特尔® 驱动程序和支持助理自动检测并安装驱动程序（推荐）

下载英特尔® 驱动程序和支持助理。有关更多信息，请参阅英特尔® 驱动程序和支持助理常见问题解答。

方法 2：手动从英特尔下载中心安装驱动程序

请执行以下步骤。如果您需要更多帮助，请参阅本页底部附近的示例文本。

  1. 识别英特尔® 处理器编号，并记下英特尔® 处理器（例如：i7-1165G7）。
    • 按键盘上的 Windows 键，然后开始输入系统，选择系统信息。这将显示处理器信息，例如处理器编号。
  2. 转到下载中心，然后在搜索栏中输入处理器编号（在第 1 步中确定）。
  3. 点击驱动程序（位于筛选条件下，左侧），然后选择所需的驱动程序结果。如果可用，我们建议选择英特尔显卡 - Windows* DCH 驱动程序以获取最新的驱动程序更新，但也可能有其他可用选项。
  4. 点击下载，并打开 .exe 文件。
  5. 按步骤执行安装，直到安装程序显示安装完成。
  6. 运行 .exe 文件后，请按照以下步骤验证安装：
    1. 开启设备管理器
    2. 按下键盘上的 Windows 键，然后开始输入设备管理器。
    3. 双击显示适配器，然后单击英特尔显卡设备。
    4. 单击驱动程序选项卡。
    5. 验证驱动程序版本和驱动程序日期是否正确。

 

示例：如何从英特尔下载中心手动安装 Windows DCH 驱动程序

以下是如何为采用英特尔酷睿处理器 i7-1165G7 的系统安装英特尔® 显卡驱动程序的分步示例。

  1. 识别英特尔® 处理器编号，并记下英特尔® 处理器。
    • 按键盘上的 Windows 键，然后开始输入系统，选择系统信息。这将显示处理器信息，例如处理器编号。

  2. 转到下载中心，然后在搜索栏中输入处理器编号（在第 1 步中确定）。

  3. 点击驱动程序（位于筛选条件下，左侧），然后选择所需的驱动程序结果。如果可用，我们建议选择英特尔显卡 - Windows* DCH 驱动程序以获取最新的驱动程序更新，但也可能有其他可用选项。

  4. 点击下载，并打开 .exe 文件。

  5. 安装程序将显示在屏幕上。点击开始安装。

  6. 请按照步骤进行安装，直到安装程序显示安装完成。 

  7. 运行 .exe 文件后，请按照以下步骤验证安装：
    1. 按下键盘上的 Windows 键，然后开始输入设备管理器。
    2. 双击显示适配器，然后单击英特尔显卡设备。

  8. 单击驱动程序选项卡。

  9. 验证驱动程序版本和驱动程序日期是否正确。

如果您在尝试上述步骤后安装显卡驱动程序出现问题，请尝试我们的英特尔® 显卡驱动程序故障排除向导，了解有关如何解决这些问题的分步说明。  您还可以联系英特尔支持来获取更多帮助。
