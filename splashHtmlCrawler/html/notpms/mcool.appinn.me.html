特性

  1. 绿色，纯净，开源软件 (Delphi, MIT 开源协议)。运行环境：PC Windows XP / 7 / 8 / 10，简体 / 繁體 / English / Unicode
  2. 极简主义，全透明无边框，“没有界面，只有音乐”。
  3. 专注本地无损音乐播放。采用 Bass 库，提供极佳音质和强大扩展能力。模块化代码设计，可高度自定义。
  4. 支持轻媒体库、歌词、可视化。最小化到托盘，占用资源少，适合边听音乐边工作。
  5. 这些功能来自网友建议：Tag 编辑、自动切换列表、播放队列、鼠标手势、均衡器、音频设备选择、全局音量滚轮（托盘区域）、字体设置、极简模式、鼠标穿透、嵌入桌面、透明调节、简单布局。
  6. 本地从有损音乐到无损音乐全格式：APE / FLAC / WavPack / MP3 / OGG / TTA / TAK / Musepack / AAC / AC3 / WMA / Wav / CD / ALAC / Aiff / MOD / CUE（通过插件支持视频格式）
