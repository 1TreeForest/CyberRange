最好看的喜剧电影（十部经典喜剧电影推荐）

作为一个电影爱好者，感觉给人最轻松的电影就是喜剧电影了，看过上百部喜剧电影，但并不是每一部都很好看，也逐渐看出了各种水平。以下推荐十部本人认为经典优秀的喜剧电影，有些影片年代比较久远，不过这些之所以称为经典，因为他们的元素在日后的几十年的电影中，都能看到这些元素的重现和致敬。这些喜剧片有点追根溯源的味道，但看完绝对可以在喜剧片的格调上提升好几个level（档次）。

另外在大家想看喜剧电影的时候，可以用来参考。祝大家观影快乐！

1.《摩登时代》 豆瓣评分9.1 导演: 查理·卓别林 主演: 查理·卓别林 / 宝莲·高黛 上映日期: 1936-02-25(美国) 片长: 87 分钟


这是看卓别林大师的第一部片子，看完明白为什么大家说他是喜剧大师了。虽然最先推荐这部，如果没看过的希望你最后看这部电影，因为当你看了这部喜剧电影之后，会发现现在的喜剧电影里有很多卓别林的影子。

如果喜欢上黑白默片的观众推荐跟卓别林同一时期的导演演员、编剧，号称冷面笑匠的巴斯特·基顿 Buster Keaton。他的《将军号》、《福尔摩斯二世》等都是超级经典的喜剧片，看完你会发现里面有成龙的很多影子。不是说成龙穿越了，而是巴斯特·基顿也是一位动作片高手，成龙的很多动作设计都来源于他的影片。


2.《白头神探》1、2、3系列 豆瓣综合评分7.5（个评分8.5） 导演: 大卫·扎克 主演: 莱斯利·尼尔森 / 普瑞希拉·普雷斯利 上映日期: 1988-12-02(美国) 片长: 85 分钟


恶搞无厘头喜剧电影的先驱，更是经典之作。 适合三五好友一起观看。增进友情、爱情、激情…全片都是梗，30多年前的电影，现在看依然不过时。如果你是周星驰影迷，那无厘头鼻祖是谁，就会成为你的新疑惑。


3.《你丫闭嘴》 豆瓣评分8.0 导演: 法兰西斯·威柏 主演: 杰拉尔·德帕迪约 / 让·雷诺 / 理查德·贝里 上映日期: 2003-10-22（法国） 片长: 85 分钟


如果你对让雷诺的感觉还停留在《这个杀手不太冷》的话，这部电影绝对颠覆你的印象。说实话这个片名起的太差劲了。要不是冲着是男主的影迷，估计不会去看。我在火车上看的这部电影，成了自己的“公路喜剧片”，绝对可以快乐地消磨时光。

4.《波普先生的企鹅》 豆瓣评分7.3 导演: 马克·沃特斯 主演: 金·凯瑞 / 卡拉·古奇诺 / 奥菲利亚·拉维邦德 上映日期: 2011-06-17(美国） 片长95分钟


小企鹅加持的经典合家欢电影，看完都想养企鹅了。

5.《憨豆的黄金周》 豆瓣评分：7.8 导演: 史蒂夫·班德莱克 主演: 罗温·艾金森 / 马克思·鲍德里 / 艾玛·德考尼斯 上映日期: 2007-07-27(中国大陆) 片长: 90分钟



憨豆的几部电影里面，比较喜欢这一部。充分展示了公路片中的美好和纯真，街头卖艺的表演真的绝了！

6.《喜宴》 豆瓣评分8.7 导演: 李安 主演: 赵文瑄 / 归亚蕾 / 金素梅 / 郎雄 / 米切尔·利希藤斯坦 上映日期: 1993-02(柏林电影节) / 1993-08-04(美国) 片长: 106 分钟


李安父亲三部曲的第二部，（没有看其他两部的可以补上：1991年《推手》1994年《饮食男女》）在90年代，这个电影主题真的很前卫啊，放到现在来说都是很值得讨论的社会话题。电影的结局相当相当的圆满啊，称得上是喜宴。

7.《菊次郎的夏天》 豆瓣评分8.7 导演: 北野武 主演: 北野武 / 关口雄介 / 岸本加世子 / Gurêto Gidayû / 井手博士 上映日期: 1999-05-20(戛纳电影节) / 1999-06-05(日本) 片长: 121 分钟


无聊的暑假来了一场冒险，整部电影有些童话色彩但又很真实。恶搞的游戏颇具温情，看似不靠谱的菊次郎用天使之铃弥补了正南心灵受到的的创伤。

8.《阳光小美女》 豆瓣评分：8.2 导演: 乔纳森·戴顿 / 维莱莉·法瑞斯 主演: 阿比盖尔·布蕾斯琳 / 托妮·科莱特 / 格雷戈·金尼尔 上映日期: 2006-08-18(美国) 片长: 101 分钟



非同寻常的家庭，很可爱的小美女。如果看片名会以为是温情片的话，那来来来，颠覆一下吧。

9.《疯狂的石头》 豆瓣评分8.1 导演: 宁浩 主演: 徐峥 / 郭涛 / 刘桦 / 黄渤 / 连晋 上映日期: 2006-06-30(中国大陆) 98 分钟(中国大陆)


小学时候看的，提高了我看喜剧电影的水平，称得上是国产黑色幽默电影的的先驱。

10.《买凶拍人》 豆瓣评分8.3 导演: 彭浩翔 主演: 葛民辉 / 张达明 / 詹瑞文 / 樋口明日嘉 / 陈惠敏 语言: 粤语 / 法语 / 日语 上映日期: 2001-08-16(香港) 片长: 94分钟


故事题材比较新奇的香港喜剧，黑色幽默、离奇荒诞，充分体现彭浩翔导演风格的电影。
