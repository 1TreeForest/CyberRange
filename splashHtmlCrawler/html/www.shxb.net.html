    莱纳你坐啊什么梗(有哪些不得不看的爆笑喜剧电影)

    2022-02-24 01:31:57
    来源：

    　　【摘要】 01.《三傻大闹宝莱坞》导演：拉吉库马尔·希拉尼主演：阿米尔·汗 / 卡琳娜·卡普尔 / 马达范标签：励志 / 印度 / 喜剧豆瓣评分：9.

    01.《三傻大闹宝莱坞》

    导演：拉吉库马尔·希拉尼

    主演：阿米尔·汗 / 卡琳娜·卡普尔 / 马达范

    标签：励志 / 印度 / 喜剧

    豆瓣评分：9.1

    影片采用插叙的手法，讲述了三位主人公法罕、拉加与兰彻间的大学故事。近三个小时的片长，却没有一个多余的情节，剧情前后呼应之巧妙值得称奇。一部讽刺社会现象的电影都可以拍的这么轻松，喜剧里还可以有这么多闪光的慧言慧语。


    02.《触不可及》

    导演：奥利维·那卡什 / 艾力克·托兰达

    主演：弗朗索瓦·克鲁塞 / 奥玛·赛 / 安乐妮

    标签：温情 / 法国 / 喜剧

    豆瓣评分：9.1

    影片改编自真实的故事。在一次跳伞运动事故后，富有的贵族菲利普下肢瘫痪，于是高薪聘请一位全职陪护，在诸多应聘者当中，他却出人意料的选择了一位毫无经验并且刚从监狱释放的混混。两个完全不同世界的人，因为教育、知识、习惯等各种不同造成情节冲突及笑料；而彼此的影响和同化则是感动的部分。


    03.《两杆大烟枪》

    导演：盖·里奇

    主演：杰森·弗莱明 / 德克斯特·弗莱彻 / 尼克·莫兰

    标签：黑色幽默 / 喜剧 / 英国

    豆瓣评分：9.0

    该片讲述了伦敦街头混混艾德和朋友参加黑社会赌局却欠下巨款，筹划打劫一群毒贩，毒贩恰在同一天对毒品供应商实施了抢劫，意外频发，最后演变成完全不可预计的闹剧。这部片可以说是喜剧片的巅峰之作，《疯狂的石头》来源于此，观看过程小心笑炸哦。


    04.《遗愿清单》

    导演：罗伯·莱纳

    主演：杰克·尼科尔森 / 摩根·弗里曼 / 西恩·海耶斯

    标签：温情 / 美国 / 老年

    豆瓣评分： 8.5

    该片主要讲述两个患了癌症晚期的病人面对癌症为他们带来的“死刑”，在余下的日子里度过丰盛和欢乐的人生的故事。两大戏骨造就的黑色喜剧，直击人心。


    05.《大话西游》系列

    导演：刘镇伟

    主演：周星驰 / 吴孟达 / 朱茵

    标签：经典 / 喜剧 / 香港

    豆瓣评分：9.1

    提起喜剧不得不提周星驰，他几乎占据了华语喜剧的半壁江山，之后有专题推荐。《大话西游》由《月光宝盒》和《大圣娶亲》两部组成，讲述了一个跨越时空的爱情故事。



    06.《唐伯虎点秋香》

    导演：李力持

    主演：周星驰 / 巩俐 / 陈百祥

    标签：喜剧 / 香港 / 搞笑

    豆瓣评分：8.3

    星爷的成名作，无厘头搞笑。该片讲述了江南才子唐伯虎对华太师府上的丫鬟秋香一见钟情，在船夫的帮助下他施展计谋混进了华府。经过一番曲折，唐伯虎终于与秋香拜堂成婚。


    07.《射雕英雄传之东成西就》

    导演：刘镇伟

    主演：梁朝伟 / 张国荣 / 张学友

    标签：喜剧 / 香港 / 搞笑

    豆瓣评分：8.7

    让你乐翻天的闹剧，光是演员阵容就很逆天，百看不厌。该片主要讲述了东邪、西毒、南帝、北丐、周伯通等人在年轻时候发生的一系列荒诞故事。
